# みなと個別指導塾 LP制作

## プロジェクト概要
みなと個別指導塾の問い合わせ獲得に特化したランディングページ。
インスタグラムのプロフィールURLとして使用するため、**スマートフォン閲覧に完全特化**する。

## 成果物
- `index.html` 1ファイルで完結（HTML + CSS + JS all-in-one）
- 外部依存はGoogleフォント・Font Awesomeのみ

## 各種制度（LP上で強調する内容）

### 成績保証制度（最大の差別化ポイント）
入塾時の学力に応じて成績向上を保証：
- 60点未満で入会 → 20点UP保証
- 60点以上80点未満で入会 → 80点台保証
- 80点以上で入会 → 90点台保証

※みなとからご提案したコース限定
※宿題忘れ等「生徒規約」を守れなかった場合は適用外
※保証条件の達成まで2学期間の猶予あり
※講習会のみのご参加は対象外

### 特待生制度
「すでに理解している分の授業料を免除する」制度：
- A特待：5科素内申23点 or 9科素内申41点 → 週2回コースが週1回料金
- S特待：5科素内申25点 or 9科素内申44点 → 週2回コースが授業料無料

### 兄弟・姉妹割引制度
- 入会金無料
- 入会時に商品券2,000円分贈呈
- 設備費お一人分全額免除
- 授業料毎月10,000円引き（兄弟・姉妹在籍中）

## UI・デザイン要件

### 固定フローティングボタン（必須）
- 画面下部に常時表示される「お問い合わせはこちら」ボタン
- タップするとお問い合わせセクションにスムーズスクロール
- iPhoneのセーフエリア対応：`padding-bottom: env(safe-area-inset-bottom)`

### スマートフォン特化
- ベースwidth: 390px（iPhone 14 Pro基準）
- タップターゲット: 最小44×44px
- フォントサイズ: 本文16px以上
- デスクトップは最大幅480pxで中央寄せ

### デザインの方向性
- **信頼感・実績感**を前面に出すデザイン
- カラーパレット：ネイビー（#1a3a5c）をメインに、アクセントカラーはゴールド（#f0a500）
- フォント：Google Fonts から「Noto Sans JP」（本文）＋「Noto Serif JP」（見出し）
- Font Awesomeアイコン使用
- スクロール時のフェードインアニメーション（IntersectionObserver使用）

### 写真・ロゴ素材
- ロゴ: `/assets/logo.png`
- ※ファイルが存在しない場合はプレースホルダーで代替

## 技術仕様
- HTML5 / CSS3 / Vanilla JS
- 外部ライブラリ不使用（Google Fonts・Font Awesomeのみ）
- フォーム送信はダミー（`action="#"`）でOK。後でGAS連携に差し替え可能にする
- `<meta name="viewport" content="width=device-width, initial-scale=1">`
- OGP・メタタグ設定済み（SNSシェア対応）
